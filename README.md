# Pixeldrain Video Downloader Extension

![Icon](logo.png)

A lightweight Google Chrome extension that allows you to easily bypass the Pixeldrain viewer and download videos/files directly using the native browser download manager.

## ✨ Features
*   **Direct Downloads:** Bypasses the Pixeldrain viewer UI to start downloading immediately.
*   **Auto-detect Links:** Automatically grabs the Pixeldrain URL from your current active tab.
*   **Fast & Lightweight:** Built purely with Vanilla JavaScript—no heavy frameworks or trackers.
*   **Privacy Focused:** All processing happens locally in your browser.

## 🚀 How to Install (Developer Mode)

Because this extension interfaces directly with Pixeldrain's content delivery network, it is not available on the Chrome Web Store. You can install it locally in seconds:

1.  Click the green **"Code"** button on this page and select **"Download ZIP"**.
2.  Extract the downloaded `.zip` file to a folder on your computer.
3.  Open Google Chrome and navigate to `chrome://extensions/`.
4.  Toggle the switch for **"Developer mode"** in the top-right corner to **ON**.
5.  Click the **"Load unpacked"** button in the top-left corner.
6.  Select the folder you extracted in Step 2.
7.  *Done!* You can now click the puzzle piece icon in Chrome to pin the extension to your toolbar.

## 🛠️ Usage
1.  Navigate to any Pixeldrain link (e.g., `https://pixeldrain.com/u/ID`).
2.  Click the extension icon. The URL will be automatically populated.
3.  Click **"Download Video"**.
4.  The file will immediately begin downloading!
